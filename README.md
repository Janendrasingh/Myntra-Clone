# Myntra Clone 👕🛍️

A modern and responsive Myntra Clone built with **React** as part of my frontend development journey. This project helped me strengthen my understanding of React by implementing various concepts learned throughout my course and applying them to a real-world e-commerce application.

## 🚀 Technologies Used

- React
- JavaScript (ES6+)
- HTML5
- CSS3
- Bootstrap
- React Hooks
- React Router
- Git & GitHub

## 📚 React Concepts Practiced

- Functional Components
- JSX
- Props
- State Management (`useState`)
- Side Effects (`useEffect`)
- Context API
- React Router
- Conditional Rendering
- Event Handling
- Lists & Keys
- Forms & Controlled Components
- Component Reusability
- Folder Structure & Code Organization

## 🎯 Objective

The goal of this project is to apply the React concepts I've learned by building a real-world e-commerce application with a clean UI, reusable components, and scalable project structure.

> This project is created for educational purposes to enhance my React development skills.
